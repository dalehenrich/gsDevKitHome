## Register Tutorial Project

```Shell
$GS_HOME/tutorials/indexing/createProjectEntry <stone-name>
```

## Install Tutorial

```Shell
$GS_HOME/tutorials/indexing/installTutorial <stone-name>
```

## Getting Started
The tutorial is driven by the execution of tODE shell scripts. 
The lessons are numbered from 1 to n and should be taken in order.
To get started with the first lesson, type the following into your tODE console:

```Shell
cd /home/tutorials/indexing
./lesson_01 --help
```
